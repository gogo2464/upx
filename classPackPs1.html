ps1 exe looks like this:

  1. <header> 2048 bytes
  2. <body> plain binary

header: contains the ps1_exe_t structure 188 bytes at offset zero rest is filled with zeros to reach the required cd mode 2 data sector size of 2048 bytes body: contains the binary data / code of the executable reqiures: executable code must be aligned to 4 must be aligned to 2048 to run from a CD optional: not aligned to 2048 (for console run only)
